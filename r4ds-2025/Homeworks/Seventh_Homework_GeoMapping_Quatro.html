Просторовий аналіз умов праці в Україні

Статистика праці за областями (2015-2019 рр.)

Author

Mykola Utkin

Вступ

Цей аналіз досліджує просторовий розподіл умов праці в областях України з використанням офіційної статистики праці за 2015-2019 роки. Ми розглядаємо частку працівників, які працюють у шкідливих умовах, та візуалізуємо регіональні диспропорції за допомогою хороплетичних карт.

Цілі дослідження:

  1. Скласти карту відсотка працівників у шкідливих умовах праці за областями

  2. Визначити регіональні закономірності та диспропорції в безпеці праці

  3. Проаналізувати часові зміни між 2015 та 2019 роками

  4. Візуалізувати промислову концентрацію та її зв’язок з умовами праці

work_cleaned <- work_data |>
  filter(
    str_detect(Показник, "Частка"), 
    `Територіальний розріз` != "Україна", 
    `Вид економічної діяльності` == "Усього",
    Стать == "Не застосовується" | is.na(Стать)
  ) |>
  select(
    oblast = `Територіальний розріз`,
    year_2015 = `2015`,
    year_2017 = `2017`,
    year_2019 = `2019`
  ) |>
  distinct() |>
  mutate(
    year_2015 = as.numeric(year_2015),
    year_2017 = as.numeric(year_2017),
    year_2019 = as.numeric(year_2019)
  ) |>

  filter(!is.na(year_2015) | !is.na(year_2017) | !is.na(year_2019)) |>
  pivot_longer(
    cols = starts_with("year_"),
    names_to = "year",
    values_to = "percentage",
    names_prefix = "year_"
  ) |>
  mutate(year = as.numeric(year)) |>
  filter(!is.na(percentage)) |>
  group_by(oblast, year) |>
  summarise(percentage = mean(percentage, na.rm = TRUE), .groups = "drop")

Інтерпретація:

Карта 2019 року демонструє значні регіональні відмінності в безпеці праці по всій Україні:

  • Промислові регіони (Дніпропетровська, Донецька, Запоріжжя) демонструють найвищий відсоток працівників у шкідливих умовах, що відображає концентрацію важкої промисловості
  • Західні області (Львівська, Івано-Франківська, Тернопільська) загалом мають нижчі відсотки, що свідчить про різні економічні структури
  • Центральні регіони демонструють помірний рівень, з варіаціями залежно від місцевого промислового складу
  • Відсутність даних для Криму та Севастополя відображає геополітичну ситуацію після 2014 року

change_data <- work_cleaned |>
  filter(year %in% c(2015, 2019)) |>

  group_by(oblast, year) |>
  summarise(percentage = mean(percentage, na.rm = TRUE), .groups = "drop") |>
  pivot_wider(
    names_from = year,
    values_from = percentage,
    names_prefix = "year_"
  ) |>

  mutate(
    change = year_2019 - year_2015,
    change_pct = (year_2019 - year_2015) / year_2015 * 100
  ) |>
  filter(!is.na(change))

Інтерпретація:

Карта змін у часі показує:

  • Покращення (синій): Кілька областей зменшили частку працівників у шкідливих умовах, можливо, через модернізацію, суворіші правила або економічну реструктуризацію
  • Погіршення (червоний): У деяких регіонах спостерігалося збільшення, що може відображати промислове розширення, покращення збору даних або зниження стандартів безпеки
  • Регіональні закономірності: Промислові східні регіони демонструють неоднозначні результати, тоді як західні регіони загалом зберегли або покращили умови

Ключові висновки:

  1. Регіональні відмінності: Існують значні відмінності між областями, причому промислові регіони демонструють у 2-3 рази вищі показники, ніж сільськогосподарські/орієнтовані на сферу послуг регіони.

  2. Промислова спадщина: Східні та центральні промислові області (Дніпропетровська, Донецька, Запорізька) постійно посідають найвищі місця за шкідливими умовами праці, що відображає концентрацію важкої промисловості.

  3. Часові тенденції: Між 2015 і 2019 роками більшість регіонів продемонстрували незначні покращення, хоча деякі зазнали погіршення, можливо, через економічний тиск.

  4. Географічні закономірності: Існує чіткий градієнт схід-захід, причому західні області загалом забезпечують безпечніше середовище праці.
